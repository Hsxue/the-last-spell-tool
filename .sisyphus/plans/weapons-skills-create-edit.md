## 工作目标

### 核心目标
实现完整的武器/技能创建和编辑功能，替换所有占位符。

### 具体交付物
- `NewWeaponDialog.tsx` - 新建武器对话框
- `NewSkillDialog.tsx` - 新建技能对话框
- `WeaponEditorForm.tsx` - 完整武器属性表单
- `SkillEditorForm.tsx` - 完整技能属性表单
- `ConfigTab.tsx` - 配置界面
- 所有功能集成到现有组件

### 完成定义
- [ ] 点击"新建"按钮打开带表单的对话框
- [ ] 表单提交后新武器/技能出现在树形视图
- [ ] 所有属性可编辑并保存
- [ ] 复制/删除功能正常工作
- [ ] `npm run build` 无错误

### 必须有
- 完整的武器属性编辑 (级别 0-5、伤害、价格、标签等)
- 完整的技能属性编辑 (效果类型、成本、范围、目标等)
- 表单验证 (必填字段、数值范围)
- 撤销/确认删除
- 配置界面管理类别和标签

### 不能有 (防护栏)
- 不能使用外部表单库 (用受控组件)
- 不能添加新依赖
- 不能修改现有 XML 解析器接口
- 不能破坏现有导入/导出功能
